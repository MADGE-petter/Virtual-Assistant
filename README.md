# Virtual Assistant

Một ứng dụng trợ lý ảo với giao diện PyQt6 và các tính năng quản lý.

## Tính năng

- 🤖 Trợ lý ảo thông minh
- 👥 Quản lý users
- 💬 Quản lý conversations
- 📊 Thống kê và báo cáo
- 💾 Database management
- ⚙️ Cài đặt hệ thống

## Cài đặt

1. Clone repository:
```bash
git clone https://github.com/MADGE-petter/Virtual-Assistant.git
cd Virtual-Assistant
```

2. Cài đặt dependencies:
```bash
pip install PyQt6
```

3. Chạy ứng dụng:
```bash
python login.py
```

## Sử dụng

- Login với tài khoản admin
- Sử dụng các tab trong admin panel để quản lý
- Tương tác với trợ lý ảo

## Database

- Sử dụng SQLite database
- File: `conversations.db`
- Auto backup hàng ngày
